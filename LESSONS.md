## 🔴 错误 #3: 飞书文档空白

**时间:** 2026-03-08 21:03

**问题:**
- 用 `create` action 创建飞书文档,传入 `content` 参数
- 文档创建成功,但内容是空的
- 姐姐打开看到空白报告

**原因:**
- 飞书文档 API 的 `content` 参数不是直接渲染 Markdown
- 需要用特定的文档块结构
- 直接传文本只能创建空文档

**解决:**
```python
# ❌ 错误方式
feishu_doc(action="create", content="Markdown内容")

# ✅ 正确方式
feishu_doc(action="create", title="标题")  # 先创建空文档
feishu_doc(action="write", doc_token="xxx", content="Markdown内容")  # 再写入
```

**教训:**
> **飞书文档操作流程:**
> 1. 用 `create` 创建文档 (只传 title)
> 2. 用 `write` 写入内容 (传 doc_token + content)
>
> **不要在 `create` 时传 `content`!**

## 🎯 核心概念

**什么是微服务 API？**
- 单一功能的自动化服务
- 通过 HTTP 请求调用
- 每次 x402 自动收费
- 高度可扩展

**为什么适合 Automaton？**
- ✅ 无需域名（使用 Conway 子域名）
- ✅ 立即启动（0 成本）
- ✅ 按需扩展（无限并发）
- ✅ 现金流稳定（即付即得）

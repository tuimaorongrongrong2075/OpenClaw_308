## 📊 技术要点

### OpenClaw 配置
- **groupPolicy**: `allowlist` vs `open`
- **groupAllowlist**: 用户 ID 白名单
- **groups.<chat_id>**: 群组特定配置
  - `enabled`: true/false
  - `requireMention`: true/false

### ClawHub 使用
- 搜索: `clawhub search <query>`
- 安装: `clawhub install <slug>`
- 卸载: `clawhub uninstall <slug> --yes`
- 列表: `clawhub list`
- 检查: `clawhub inspect <slug>`

### 前端设计原则 (Frontend Skill 1.0.2)
- **色彩**: 70-20-10 规则，深色主题避免纯黑
- **字体**: 避免通用字体 (Inter, Roboto, Arial)
- **排版**: 2x+ 尺寸跳跃，创造视觉层级
- **动画**: CSS 优先，GPU 加速
- **性能**: 懒加载，代码分割

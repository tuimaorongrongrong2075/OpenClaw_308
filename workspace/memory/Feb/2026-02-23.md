## 知识问答（姐姐学习工具）
- **mpm** - 可能是 Multi-Package Manager（需确认）

- **brew** - Homebrew，macOS/Linux 包管理器
- **curl** - 命令行 HTTP 工具，支持多种协议

### 关键区别
- **Homebrew vs npm**: Homebrew 管理系统级软件，npm 管理 JS 包

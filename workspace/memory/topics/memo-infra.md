## Context Sync vs Memo Infra 精准区别

### Context Sync（事件记忆）
| 维度 | 说明 |
|------|-------|
| **性质** | 即时动作（动词） |
| **触发** | 手动（你说"记住"、"记一下"） |
| **记录** | 具体事件、当天对话内容 |
| **时效** | 有时效（那天的事） |
| **文档** | `memory/topics/` + `memory/YYYY-MM-DD.md` |
| **用途** | 下次继续这个话题 |

**例子**：今天姐姐问了 GitHub star 和 fork 的区别 → 记录下来，下次可以继续

### Memo Infra（知识记忆）
| 维度 | 说明 |
|------|-------|
| **性质** | 长期架构（名词） |
| **触发** | 定时（每周日 cron） |
| **记录** | 关系图谱、实体、偏好 |
| **时效** | 长期有用（任何时候） |
| **文档** | `entities/` + `relationships/` + `facts/` |
| **用途** | 任何时候都知道她的爱好、关系 |

**例子**：姐姐在学 FL Studio → 任何时候都知道她的爱好

---

## 一句话总结

- **Context Sync** = 记录具体事情（事件）
- **Memo Infra** = 记录关系图谱（知识）

- **教训:** 配置文件头部必须有核心信息!

### 错误 2: 工作目录结构混乱
- **问题:** 旧模板文件混杂在新文件中,导致误解
- **原因:** 没有清理旧文件就开始工作
- **解决:** 删除 238 个旧文件,保持目录清爽
- **教训:** 先清理再工作,保持环境整洁!

### 错误 3: 飞书文档空白
- **问题:** 用 `create` 创建文档时直接传 `content` 参数,结果文档是空的
- **原因:** 飞书文档 API 的 `content` 参数需要特定格式,Markdown 不会自动渲染
- **解决:** 先创建文档,再用 `write` 写入完整内容
- **教训:** 飞书文档要用 `write` action,不能用 `create` + `content`!
